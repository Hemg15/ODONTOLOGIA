ARTICULOS

Espejo Intraoral con mango
  • Para qué sirve: Permite visualizar zonas de difícil acceso y separar los tejidos blandos (mejillas y lengua).
  • Precio estimado: $120 - $250 MXN.
 Explorador Dental de dos extremos
  • Para qué sirve: Se usa para detectar caries, sarro y revisar la integridad de las restauraciones.
  • Precio estimado: $150 - $300 MXN.
 Pinza Algodonera
  • Para qué sirve: Sirve para sujetar y transportar algodones, gasas o pequeños materiales a la boca.
  • Precio estimado: $180 - $350 MXN.
Pieza de Alta Velocidad (Turbina)
  • Para qué sirve: Instrumento rotatorio para eliminar tejido duro (caries) y preparar cavidades.
  • Precio estimado: $1,500 - $4,500 MXN.
 Micromotor con Contra-ángulo
  • Para qué sirve: Se usa a baja velocidad para pulir resinas, realizar limpiezas o desgastar prótesis.
  • Precio estimado: $2,800 - $6,000 MXN.
 Lámpara de Fotocurado LED
  • Para qué sirve: Emite una luz azul especial para endurecer (polimerizar) las resinas y selladores.
  • Precio estimado: $1,200 - $3,500 MXN.
Kit de Resinas Compuestas
  • Para qué sirve: Material estético para realizar "tapaduras" o restauraciones del color del diente.
  • Precio estimado: $2,500 - $5,000 MXN (el kit completo).
 Alginato para Impresiones
  • Para qué sirve: Polvo que se mezcla con agua para sacar moldes de la boca del paciente.
  • Precio estimado: $250 - $450 MXN (bolsa de 450g).
 Ácido Grabador (Jeringa)
  • Para qué sirve: Prepara la superficie del esmalte para que la resina se adhiera correctamente.
  • Precio estimado: $150 - $300 MXN.
Adhesivo Dental
  • Para qué sirve: Actúa como el "pegamento" entre el diente preparado y la resina.
  • Precio estimado: $800 - $1,600 MXN (frasco pequeño).
 Anestesia Local (Caja de cartuchos)
  • Para qué sirve: Bloquea la sensibilidad en la zona a tratar para que el paciente no sienta dolor.
  • Precio estimado: $650 - $950 MXN (caja con 50 cartuchos).
 Barreras de Protección (Campos y eyectores)
  • Para qué sirve: Mantienen la higiene del consultorio y ayudan a succionar la saliva durante el tratamiento.
  • Precio estimado: $200 - $400 MXN (paquete).
